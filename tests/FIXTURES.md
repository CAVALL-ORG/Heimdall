# Fixture provenance ledger

Every image shipped in the Heimdall test tree, with its origin and
license / clearance. Heimdall ships only fixtures cleared as own-work,
public-domain, or otherwise carrying no third-party copyright.

Coverage at curation time: **109 images**, each provenanced via a
per-file `<image>.LICENSE.txt` sidecar, the
`tests/ketcher/image-to-smiles/image_sources.jsonl` index, or a
directory-group declaration in `tests/scientific/images/SOURCES.md`.
No image is unprovenanced.

Notes on the two clearance classes:

- **Public domain** — Wikimedia Commons skeletal renderings or PubChem
  2D depictions used as paper-style figure stand-ins. Each carries a
  `<image>.LICENSE.txt` sidecar and/or an `image_sources.jsonl` row with
  the source URL and citation.
- **Own work / no third-party copyright** — generated by this project's
  own pipeline: Ketcher / Indigo renders of input or canonical SMILES, or
  PIL-synthesized control panels. The `diverse/` set is 100%
  Ketcher-rendered from PubChem-sourced input SMILES (CIDs cited in the
  manifest `notes`). The `tests/panels/cropped/` images are
  whitespace-trimmed copies of the same-named shared fixtures (so a
  trimmed copy inherits the clearance of its source); they exist only as
  typeset-panel companions.

The authoritative per-image origin/license records remain the
`<image>.LICENSE.txt` sidecars and the two index files
(`tests/scientific/images/SOURCES.md`,
`tests/ketcher/image-to-smiles/image_sources.jsonl`); this ledger is a
generated roll-up of them.
